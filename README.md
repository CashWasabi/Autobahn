# Autobahn

```
  o----------------o
  |                |
  |      /||\      |
  |     /_||_\     |
  | =m==========m= |
  |   /   ||   \   |
  |  /____||____\  |
  |                |
  o----------------o
```

A simple spinning thread pool implementation.

## Usage with internal memory allocation

Using `Autobahn` is easy.
It's so small you can just copy and paste it into your project.

## Features

- low latency thread pool
- lock free threading
